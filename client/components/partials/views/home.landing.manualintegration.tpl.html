Last mile services using client panel

  • Using given login id and password , you can login to cl.delhivery.com and upload package using upload package button on the left panel .
  • Fill in all the necessary details and the submit button would lit up (This button will not lit up in case you have left any of the fields blank).
  • In order to select pickup location , you need to write the initials of the pickup location (warehouse name) in the given box and a pop will appear , select that pop up and hit the submit button .
  • Once the package is pushed you will get a waybill no. against that package .
  • Now navigate to cl.delhivery.com/package/ ,click on print packing slip. This opens the packing slip in a pop up window (make sure pop up is not blocked for cl)
  • In order to ensure that the pickup happens after manifesting the package details into our system you need to send a mail to vendordesk@delhivery.com and dispatch@delhivery.com with the below mentioned details for each pickup location:
    • A. Location of pickup along with accurate address
    • B. Date/Time of pickup
    • C. Count of packages to be picked up for the particular location
